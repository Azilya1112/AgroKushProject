#Agro-Kush Web application
The goal is to facilitate the work of tracking the work performed in each department of the workshop. The work can be divided into planned, work in progress, completed. A detailed understanding of the plans, tasks and the work of the workshop itself will help to track progress, as well as allow you to see a particular task in a structured and clear way.
The program also allows you to keep records of equipment (list, characteristics) and perform some operations (deletion, addition, modification)
